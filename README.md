# Snowsock

Simulate watching a Redis cluster for stores not updating

## Prerequisites

1. Docker

## Installation

1. `git clone https://github.com/JeffreyMFarley/snowsock.git`

## Running

1. `cd snowsock/services`
1. `docker compose up`
    1. The three services will run
    1. `^C` to exit
1. `docker compose down`
